<--------  Mode d'emploi :  --------> 

- Lancement du jeu ( avec les commandes dans le terminal )
- Selection du Mode de Jeu
	- Jouer
		- Humain
		- Algorithme
		- retour Menu
	- Niveau
		- Niveau 1
		- Niveau 2
		- Niveau 3
		- Retour Menu

<--------  Humain :  --------> 

- Pour jouer utiliser les touches "z q s d"

- Pour relancer le niveau, importer un niveau, relancer menu (non implémenté) aller dans option en haut à gauche 

- Niveau réussi cliquer sur Next Level en bas a droite

- Vous pouvez voir votre score en bas (nombre de deplacement)

<--------  Algorithme :  --------> 

- Choisir l'algorithme souhaiter 
	- Greedy 
	- A Star

- Séléctioner un niveau ( seulement les niveaux t1 .. t6.txt au niveau de la racine src )

- Appuyer sur résoudre en haut à droite

- Appuyer sur Next pour voir les coups 1 par 1 en bas à gauche
